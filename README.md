# Эмулятор датчика температуры и контроллера светодиода
Проект написан на C++/Qt

## Описание

* Каждые 2 секунды “измеряется температура”, то есть генерируется случайное число в диапазоне 20–35 °C
* Если температура выше 30 °C — “включается светодиод”, иначе “выключается”
* Каждые 5 секунд отправляется JSON-пакет в консоль в формате {"temp": 31.2, "led": "ON"}
* Кнопки "LED ON" и "LED OFF" позволяют вручную управлять светодиодом
